About
This is a simple URL shortener built with Go, HTMX, and Tailwind CSS.
It takes a novel approach to URL shortening by using a hash of the URL as the shortened URL. This means that the same URL will always be shortened to the same value. It uses a dictionary, the index of the word in the dictionary, is hashed to a 3 character string using a custom base62 encoding.
